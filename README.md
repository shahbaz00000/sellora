🛒 E-Commerce MERN App

A full-stack E-Commerce Web Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This project includes user authentication, product management, cart functionality, and secure order processing.


🚀 Features
🔐 Authentication & Authorization
User registration & login
JWT-based authentication
Protected routes
Role-based access (Admin/User)
🛍️ Product Management
Add, update, delete products (Admin)
View product listings
Product details page
Search & filter products
🛒 Cart System
Add to cart
Remove from cart
Update quantity
Persistent cart using database/local storage
📦 Order Management
Place orders
View order history
Admin can manage all orders
💳 Payment Integration
Razorpay / Stripe integration (if implemented)
Secure checkout process
📍 Address Management
Add and update delivery address
Store user shipping details
🧱 Tech Stack
Frontend
React.js
Redux / Context API (if used)
Tailwind CSS / Bootstrap
Backend
Node.js
Express.js
Database
MongoDB (Mongoose)
Authentication
JSON Web Token (JWT)
bcrypt (for password hashing)
